# Smart Traffic Light Control using PID Controller

This project implements an adaptive traffic signal control system using a PID controller in MATLAB.

## Features
- Adaptive signal timing
- Queue length monitoring
- Traffic density heatmap
- PID vs Fixed signal comparison
- PID tuning analysis

## Technologies
- MATLAB
- Control Systems (PID)
